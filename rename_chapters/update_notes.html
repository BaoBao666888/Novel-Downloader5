Phiên bản 0.1.7 — Bản vá ổn định

Sau bản cập nhật lớn 0.1.6, chúng tôi phát hành 0.1.7 để xử lý các lỗi phát sinh khi build exe: icon nguồn không hiện, preview chia file không mở trên một số cấu hình và định dạng `{num-1}` gây crash.

Sửa lỗi quan trọng

- Icon popup chọn nguồn giờ luôn hiển thị, ngay cả khi chạy file cài đặt (app tự tìm icon trong thư mục cài hoặc PyInstaller bundle). - “Xem trước” trong tab Chia File hoạt động ổn định: double-click hàng nào cũng mở, kể cả sau khi đóng cửa sổ preview. - Định dạng tên file `{num±n}` (ví dụ `{num-1}`) đã được parser mới xử lý đúng ở cả preview lẫn khi chia file thực tế. - Combobox Regex chia/định dạng file nhớ giá trị cuối cùng – không còn hiện chuỗi rỗng khi relaunch.

Hướng dẫn cập nhật

- Nhấn Tải & Cài đặt để download bản cài đặt (.exe) tự động. - Ứng dụng sẽ đóng để tránh xung đột; bộ cài sẽ khởi chạy để cài đặt. - Nếu muốn giữ bản cũ, chọn Hủy hoặc Mở bằng trình duyệt.
