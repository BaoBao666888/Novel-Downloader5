Phiên bản 0.2.4.1 — Fanqie Downloader & Wikidich

Điểm mới

- Download Novel 5: thêm reset bridge, cấu hình timeout, template tên file ({title}_{author} mặc định), tự điền URL Fanqie khi bấm Download từ Liên kết (dùng thư mục liên kết, không ghi config), cho chỉnh thư mục lưu tạm thời. - Liên kết trang truyện: nút Download mở ND5 với thư mục liên kết; giữ link auto update để sẵn URL. - Upload Wikidich: yêu cầu chọn tối thiểu 2 file; cảnh báo file nhỏ hiển thị đỏ trong log UI và ghi vào file log. - Auto update Fanqie: dừng nếu thiếu nội dung, tối thiểu 2 chương mới; không tạo file khi thiếu; tự thêm Credit nếu bật; kiểm tra file nhỏ; chọn template tên chương chuẩn. - ND5 lưu log ngày theo thư mục logs, tự dọn log cũ; thêm delay request cấu hình được.

Wikidich

- Thêm chế độ “Chỉ đồng bộ số chương” khi tải chi tiết: chỉ lấy Works, không tải văn án. - Đồng bộ số chương chỉ tăng; nếu server ít hơn local thì giữ số cũ và tô cam (cả ngày cập nhật), không giảm dữ liệu. - 404: đánh cờ đỏ, popup danh sách 404 (double-click mở trình duyệt), tô đỏ ưu tiên hơn “New”, lưu dấu vết để tô lại sau refresh. - Kiểm tra cập nhật: reset màu/flag đúng; cập nhật cột Cập nhật chỉ giữ phần ngày.

Sửa lỗi & tối ưu

- Ghi cache sau mỗi truyện khi tải chi tiết/kiểm tra để tránh mất dữ liệu nếu dừng giữa chừng. - DS Chương đánh số tuần tự theo thứ tự server. - Giảm lỗi timeout bridge bằng timeout cấu hình; log rõ lỗi bridge/batch. - Chỉnh lại giao diện ND 5.

Hướng dẫn cập nhật

- Nhấn Tải &Cài đặt để lên 0.2.5 (ứng dụng sẽ tự đóng trước khi chạy bộ cài). - Nếu muốn xem chi tiết, chọn Mở bằng trình duyệt.
