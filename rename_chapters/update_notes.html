Phiên bản 0.3.3.4 - TextOps và OCR dễ dùng hơn

Xử lý văn bản

  • Mở file gần nhất trong menu Xử lý văn bản giờ mở thẳng file recent mới nhất, không bật thêm cửa sổ chọn trùng với Lịch sử.
  • Chia file mở dạng cửa sổ riêng, có bảng preview cuộn dọc/ngang, hiển thị tên file sẽ tạo và chữ lớn hơn để dễ nhìn chương.
  • Double-click một phần trong preview chia file hoặc bấm Sửa raw... để mở cửa sổ xem/sửa raw riêng cho phần đó; khi đóng nếu có sửa app sẽ hỏi lưu vào preview.
  • Công cụ nhanh mở dạng cửa sổ riêng, có vùng cuộn để dùng tốt hơn trên màn hình thấp.
  • Bỏ dòng thông tin file/encoding lặp ở đầu tab tài liệu; status bar cuối cửa sổ vẫn hiển thị path, encoding đọc, encoding lưu và trạng thái lưu.
  • Tự nhận diện thêm UTF-16/Unicode khi mở file txt; nếu nội dung có dấu hiệu sai encoding, app mở bảng chọn encoding có preview để user chọn trực quan.
  • Lưu file mặc định bằng UTF-8 để tránh lỗi Unicode; có thể đổi mã lưu trong File > Chọn mã lưu... khi cần.
  • Hộp Tìm và Tìm & thay thế có ghim regex riêng; regex ghim ở tab nào chỉ ưu tiên trong tab đó.
  • Tab tài liệu vẫn dùng tên file, chữ tab rõ và đậm hơn.

OCR ảnh

  • Tách OCR khỏi tab Dịch thành cửa sổ riêng trong menu OCR > Mở cửa sổ OCR; có thể nạp một hoặc nhiều ảnh, xem ảnh bên trái và text/kết quả bên phải.
  • Chuyển OCR sang PaddleOCR runtime tách riêng; app chính chỉ gọi tools/ocr_runtime/ocr_runtime.exe để giữ app nhẹ và dễ cập nhật runtime riêng.
  • Khi mở OCR, cửa sổ hiện trước rồi mới kiểm tra runtime trong nền; trong lúc runtime đang load app sẽ chặn OCR để tránh đơ UI.
  • Nếu runtime chưa có hoặc bản online mới hơn bản đang cài, app sẽ hỏi tải/cập nhật runtime từ version.json trước khi OCR.
  • Thêm danh sách model theo ngôn ngữ kèm mô tả chức năng: Trung, Việt/Latin, Anh, Hàn, Nhật và phồn thể.
  • Thêm nút Tải model; user không cần tự tải file model, OCR runtime tự tải/cache model chính thức theo lựa chọn.
  • Preview ảnh có thể cuộn chuột để zoom nhanh, giữ chuột kéo để pan, và vẽ khung nhận diện OCR lên ảnh sau khi chạy xong.
  • Khi OCR đang chạy, đóng cửa sổ sẽ hỏi xác nhận; ảnh đang nằm trong phiên OCR sẽ không bị xóa nhầm bằng phím Delete/menu xóa.
  • Thêm tùy chọn Gộp dòng ngang để tự nối các box OCR gần nhau thành đoạn văn, phù hợp text ngang trái qua phải.

Hướng dẫn cập nhật

  • Nhấn Tải & Cài đặt để lên 0.3.3.4. Ứng dụng sẽ tự đóng trước khi chạy bộ cài.
  • Nếu cần xem ghi chú bằng trình duyệt, bấm Mở bằng trình duyệt.
