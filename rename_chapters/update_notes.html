Phiên bản 0.3.3 - Tối ưu và sửa lỗi

Điểm mới & Sửa lỗi

Hiệu năng / Đóng gói

- Tăng tốc độ khởi động app rõ rệt. - Tối ưu đóng gói để giảm dung lượng ứng dụng.

Trình duyệt Qt / ND5

- Thêm cơ chế fetch HTML qua trình duyệt Qt (IPC FETCH_HTML) - Thêm API host cho plugin ND5 gọi browser fetch. - ND5 hỗ trợ tích hợp tải bằng ext/app VBook khi Reader Server đang chạy.

Đọc truyện

- Tối ưu tab Đọc truyện: - Sửa lỗi có lúc không dừng được Reader Server. - Bổ sung log trực quan hơn để theo dõi trạng thái server và thao tác quản lý. - Tích hợp tải truyện từ ext của app VBook vào hệ thống Reader. - Tối ưu hiệu năng khởi động Reader Server và thay đổi cấu trúc build. - Fix lỗi không lưu config Reader Server khi tắt app.

Wikidich / Koanchay

- Cửa sổ Tải chi tiết thêm tùy chọn: - Quét tên quyển trong phạm vi tải chi tiết (chỉ hoạt động với works chính chủ). - Bổ sung khu hiển thị Tên quyển hiện có ngay dưới khung Văn án. - Bộ lọc cơ bản thêm ô Tên quyển để tìm nhanh theo tên volume đã quét. - Khi dùng Tải chi tiết cùng lúc với Quét tên quyển, hệ thống sẽ chạy tuần tự và lưu dần để có thể tiếp tục nếu bị gián đoạn. - Khi mở trang Chỉnh sửa hoặc chạy Auto Update, app sẽ tận dụng luôn dữ liệu trang sửa để đồng bộ tên quyển về local. - Sửa lỗi nút Chỉnh sửa không hoạt động với Koanchay.

Hướng dẫn cập nhật

- Nhấn Tải & Cài đặt để lên 0.3.3 (ứng dụng sẽ tự đóng trước khi chạy bộ cài). - Nếu muốn xem chi tiết, chọn Mở bằng trình duyệt.
