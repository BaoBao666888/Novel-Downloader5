Phiên bản 0.3.2 - Big update (Fanqie/Reader/Auto Update)

Điểm mới

Fanqie & ND5: - Big update: dùng logic Fanqie theo app mới nhất. - Điều chỉnh kiểm tra cập nhật Fanqie sang API mới. - Auto Update tự đồng bộ số chương trước rồi mới update theo Fanqie. - Thêm cache cho truyện tải từ ND5. Wikidich/Koanchay: - Thêm điều chỉnh domain trong Cài đặt request và tự update domain trong dữ liệu local. - Thêm chỗ lưu riêng cho các truyện đồng quản lý khi tải work (tránh tải lại hết mất thời gian). - Thêm Auto Update tự động toàn cục. Đọc truyện (Web app): - Thư viện: nhập file, hiển thị các truyện đã nhập. - Thông tin book: mục lục, thông tin sách, chỉnh sửa, name riêng. - Trang đọc: giao diện web cơ bản như các web đọc truyện phổ biến. Xóa rác: - Tăng cường chức năng Xóa rác (gợi ý/đánh giá, diff preview, pipeline nhiều chế độ). Đổi tên: - Tối ưu chức năng regex cho tab Đổi tên. Giao diện: - Tăng tham số cho tùy chỉnh giao diện.

Sửa lỗi

- Sửa lỗi giải nén bằng app thiếu file. - Fix lỗi không nhận được cookie PO18 dù đã đăng nhập trong profile hiện tại.

Hướng dẫn thao tác (mới)

- **Auto Update toàn cục**: Tab Wikidich/Koanchay → menu **Auto Update ▾** → **Đánh dấu** truyện đủ điều kiện → **Tự động** để chạy hàng loạt → **Lịch sử** để xem kết quả. - **Đọc truyện**: vào tab **Đọc truyện** để quản lý server + UI; có thể Start/Stop server và mở giao diện đọc ngay từ app. - **Xóa rác**: menu **Công cụ → Công cụ Xóa rác...** → chọn File/Thư mục → xem **Diff** trước khi áp dụng. - **Gợi ý regex**: Tab **Đổi tên** → nút **Gợi ý** (Tên file/Nội dung) để gợi ý regex theo file đang chọn. - **Đổi domain**: Cài đặt request → đổi domain Wikidich/Koanchay; dữ liệu local sẽ tự rewrite theo domain mới. - **ND5 cache**: ND5 → **Cài đặt** → **Dữ liệu truyện** để xem/xóa cache theo từng truyện.

Hướng dẫn cập nhật

- Nhấn Tải & Cài đặt để lên 0.3.2 (ứng dụng sẽ tự đóng trước khi chạy bộ cài). - Nếu muốn xem chi tiết, chọn Mở bằng trình duyệt.
