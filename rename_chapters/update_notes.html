Phiên bản 0.3.3.4 - TextOps dễ dùng hơn

Xử lý văn bản

  • Mở file gần nhất trong menu Xử lý văn bản giờ mở thẳng file recent mới nhất, không bật thêm cửa sổ chọn trùng với Lịch sử.
  • Chia file mở dạng cửa sổ riêng, có bảng preview cuộn dọc/ngang, hiển thị tên file sẽ tạo và chữ lớn hơn để dễ nhìn chương.
  • Double-click một phần trong preview chia file hoặc bấm Sửa raw... để mở cửa sổ xem/sửa raw riêng cho phần đó; khi đóng nếu có sửa app sẽ hỏi lưu vào preview.
  • Công cụ nhanh mở dạng cửa sổ riêng, có vùng cuộn để dùng tốt hơn trên màn hình thấp.
  • Bỏ dòng thông tin file/encoding lặp ở đầu tab tài liệu; status bar cuối cửa sổ vẫn hiển thị path, encoding đọc, encoding lưu và trạng thái lưu.
  • Tự nhận diện thêm UTF-16/Unicode khi mở file txt; nếu nội dung có dấu hiệu sai encoding, app mở bảng chọn encoding có preview để user chọn trực quan.
  • Lưu file mặc định bằng UTF-8 để tránh lỗi Unicode; có thể đổi mã lưu trong File > Chọn mã lưu... khi cần.
  • Hộp Tìm và Tìm & thay thế có ghim regex riêng; regex ghim ở tab nào chỉ ưu tiên trong tab đó.
  • Tab tài liệu vẫn dùng tên file, chữ tab rõ và đậm hơn.

Hướng dẫn cập nhật

  • Nhấn Tải & Cài đặt để lên 0.3.3.4. Ứng dụng sẽ tự đóng trước khi chạy bộ cài.
  • Nếu cần xem ghi chú bằng trình duyệt, bấm Mở bằng trình duyệt.

Ghi chú các bản trước

  • 0.3.3.3 đã chuyển Xử lý văn bản thành cửa sổ riêng, tách state TextOps khỏi config.json, thêm autosave/khôi phục tài liệu và tăng bộ nhớ regex.
  • Đã cải thiện Download Novel 5: ô phạm vi tải rộng hơn, Enter để tải và tự reset phạm vi khi lấy truyện khác.
  • Đã dọn regex gợi ý cũ trong Đổi tên khi đổi cuốn, thêm lọc tag Wikidich theo nhóm, ẩn Koanchay mặc định và thêm OCR ảnh bằng Windows OCR.
  • Đã cải thiện cửa sổ ghi chú update và bộ cài update để dọn runtime cũ trước khi chép bản mới.
